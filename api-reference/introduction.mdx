## Introduction

L'API MecaPy permet de déployer et d'exécuter vos fonctions Python via des endpoints REST.

<Note>
  **Base URL** : `https://api.mecapy.com`
</Note>

## Authentification

Toutes les requêtes nécessitent un **Bearer token**.

```bash
Authorization: Bearer YOUR_API_KEY
```

Obtenez votre API key depuis le [dashboard](https://app.mecapy.com/settings/api-keys).

## Exemple rapide

### Déployer un package

```bash
curl -X POST https://api.mecapy.com/packages/from-git \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "git_url": "https://github.com/votre-username/votre-repo",
    "git_ref": "main"
  }'
```

### Exécuter une fonction

```bash
curl -X POST https://api.mecapy.com/functions/mon-package/ma-fonction/1.0.0/invoke \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "param1": "valeur1",
    "param2": 42
  }'
```

## SDK Python

Utilisez le SDK Python officiel pour simplifier vos appels :

```bash
pip install mecapy-sdk
```

```python
from mecapy import MecaPy

client = MecaPy(api_key="YOUR_API_KEY")

# Déployer un package
package = client.deploy_from_git(
    git_url="https://github.com/votre-username/votre-repo",
    git_ref="main"
)

# Exécuter une fonction
result = client.invoke(
    function_id="mon-package/ma-fonction/1.0.0",
    inputs={"param1": "valeur1", "param2": 42}
)

print(result)
```
