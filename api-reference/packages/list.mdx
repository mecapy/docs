---
title: "List Packages"
api: "GET /packages"
description: "List public packages and own packages (if authenticated)"
---

Returns a paginated list of packages. **No authentication required.**

- **Without token** — returns all public packages
- **With token** — returns all public packages + your own private packages

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number (starts at 1)
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  Items per page (max 100)
</ParamField>

<RequestExample>

```bash Public (no auth)
curl https://api.mecapy.com/packages
```

```bash Authenticated
curl https://api.mecapy.com/packages \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```python Python
# Public
response = requests.get("https://api.mecapy.com/packages")

# Authenticated (also returns your private packages)
response = requests.get(
    "https://api.mecapy.com/packages",
    headers={"Authorization": f"Bearer {token}"},
)
```

</RequestExample>
