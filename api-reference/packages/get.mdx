---
title: "Get Package"
api: "GET /packages/{package_id}"
description: "Get details of a specific package"
---

Returns complete package information including all deployed function versions.

- **Public packages** — accessible without authentication
- **Private packages** — require authentication and ownership

## Path Parameters

<ParamField path="package_id" type="string" required>
  Unique identifier of the package (UUID)
</ParamField>

<RequestExample>

```bash Public package (no auth)
curl https://api.mecapy.com/packages/17ad586c-0579-48e9-ba00-dcf6668d648c
```

```bash Private package (auth required)
curl https://api.mecapy.com/packages/17ad586c-0579-48e9-ba00-dcf6668d648c \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```python Python
response = requests.get(
    "https://api.mecapy.com/packages/17ad586c-0579-48e9-ba00-dcf6668d648c",
    headers={"Authorization": f"Bearer {token}"},  # optional for public packages
)
```

</RequestExample>
