---
title: "Deploy Function from ZIP"
api: "POST /functions/deploy"
description: "Deploy a serverless function from a base64-encoded ZIP archive"
---

Deploy your code as a serverless function by uploading a base64-encoded ZIP archive. The platform automatically manages scaling, resource allocation, load balancing, and cold start optimization.

## Request Body

<ParamField body="name" type="string" required>
  Unique function name (lowercase, hyphens allowed)
</ParamField>

<ParamField body="code" type="string" required>
  Base64-encoded ZIP archive containing function code
</ParamField>

<ParamField body="handler" type="string" required>
  Handler path (module:function format)
  
  **Example:** `main.calculate`
</ParamField>

<ParamField body="runtime" type="string" required>
  Python runtime: `python311` or `python312`
</ParamField>

<ParamField body="description" type="string">
  Function description
</ParamField>

<ParamField body="memory_limit" type="integer" default="512">
  Memory in MB (128-2048)
</ParamField>

<ParamField body="timeout" type="integer" default="300">
  Timeout in seconds (1-900)
</ParamField>

<ParamField body="environment_variables" type="object">
  Environment variables as key-value pairs
</ParamField>

<RequestExample>

```python Python
import base64
import requests

# Base64 encode ZIP file
with open("function.zip", "rb") as f:
    code_b64 = base64.b64encode(f.read()).decode()

response = requests.post(
    "https://api.mecapy.com/functions/deploy",
    headers={"Authorization": f"Bearer {token}"},
    json={
        "name": "calculate-stress",
        "runtime": "python311",
        "handler": "main.calculate",
        "code": code_b64,
        "environment_variables": {"SAFETY_FACTOR": "1.5"}
    }
)
```

</RequestExample>
